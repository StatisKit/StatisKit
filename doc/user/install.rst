Prerequisites
=============

.. The **StatisKit** software suite is composed of multiple software.
.. These software are designed to be used for statistical analyses.

In order to ease the installation of this software suite on multiple operating systems, the **Conda** package and environment management system is used.
To install **Conda**, you can execute the following commands:

*   On Linux,

	.. code-block:: bash
	
		wget http://statiskit.readthedocs.io/en/latest/user/install.sh
		source install.sh

*   On Mac Os X,

	.. code-block:: bash
	
		curl http://statiskit.readthedocs.io/en/latest/user/install.sh
		source install.sh

*   On Windows,

	.. code-block:: batch

		wget http://statiskit.readthedocs.io/en/latest/user/install.bat
		call install.bat

.. note::

	For more information concerning **Conda**, please refers to its own documentation.
